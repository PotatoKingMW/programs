## WriteDataset.exe使用说明：
* example.xlsx已内置于本程序中

## CommunityDetection.exe使用说明：

## TIF2JPG.exe使用说明：
* 将此程序与需要转换的tif或tiff格式图片放置于同一目录下即可自动识别文件
